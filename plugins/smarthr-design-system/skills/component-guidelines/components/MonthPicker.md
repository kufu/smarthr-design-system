# MonthPicker

> ⚠️ **非推奨**: MonthPickerは非推奨です。Input[type="month"]を使ってください。

ユーザーに年と月を入力させる際に使用するコンポーネントです。

## import

```ts
import { MonthPicker } from 'smarthr-ui'
```

## Props

| Props 名 | 型 | デフォルト値 | 必須 | 説明 |
|---|---|---|---|---|
| error | boolean | - | - | フォームにエラーがあるかどうか |

## 実装ルール

MonthPicker に直接関連する eslint-plugin-smarthr のルールは現時点ではありません。

## 使い方チェックリスト

使い方チェックリスト（Layer 3）は設定されていません。
