# MonthPicker

> ⚠️ **非推奨**: MonthPickerは非推奨です。Input[type="month"]を使ってください。

ユーザーに年と月を入力させる際に使用するコンポーネントです。

## import

```ts
import { MonthPicker } from 'smarthr-ui'
```
