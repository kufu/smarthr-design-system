# TimePicker

> ⚠️ **非推奨**: TimePickerは非推奨です。Input[type="time"]を使ってください。

ユーザーに時刻（時と分、任意で秒）を入力させる際に使用するコンポーネントです。

## import

```ts
import { TimePicker } from 'smarthr-ui'
```
