ウェブアクセシビリティを確保・向上させるための簡易チェックリストです。このリストに記載されている項目を満たすと、おおよそSmartHRのウェブアクセシビリティ方針に掲げた品質を達成できます。
チェックする内容によっては例外があることがあります。参考に記載したリンクも参照してください。

## 1. 代替テキスト

- [ ] [1. 画像に代替テキストが付与されている](./alternative-text/)
- [ ] [2. 装飾目的の画像が無視できるようになっている](./decorative-image/)

参考

- [画像の代替テキスト - SmartHR Design System](../alternative-text/)
- [非テキストコンテンツ - WCAG 2.0 解説書](https://waic.jp/docs/UNDERSTANDING-WCAG20/text-equiv-all.html)
- [1.1.1 画像に代替テキストを提供する - Ameba Accessibility Guidelines](https://a11y-guidelines.ameba.design/1/1/1/)

## 2. 動画・音声・アニメーション

- [ ] 1. 動画の音声に字幕が付与されている
- [ ] 2. 動画の内容を解説した音声、またはコンテンツがあるか
- [ ] 3. 音声や動画、アニメーションが自動で再生される場合、一時停止できるか
- [ ] 4. 画面内に1秒に3回以上の点滅や閃光を起こすものがない

参考

- [音声解説、又はメディアに対する代替 (収録済) - WCAG 2.0解説書](https://waic.jp/docs/UNDERSTANDING-WCAG20/media-equiv-audio-desc.html)
- [1.2.1 収録済みの動画に代替コンテンツを提供する - Ameba Accessibility Guidelines](https://a11y-guidelines.ameba.design/1/2/1/)
- [1.4.2 音声を制御できるようにする - Ameba Accessibility Guidelines](https://a11y-guidelines.ameba.design/1/4/2/)
- [2.2.2 動く、自動更新するコンテンツに配慮する - Ameba Accessibility Guidelines](https://a11y-guidelines.ameba.design/2/2/2/)
- [3 回の閃光、又は閾値以下 - WCAG 2.0解説書](https://waic.jp/docs/UNDERSTANDING-WCAG20/seizure-does-not-violate.html)

## 5. 操作しやすさ

- [ ] [1. キーボードで操作可能](./keyboard/)

  - [ ] キーボードで操作していることがわかる（フォーカス状態が見えること）
