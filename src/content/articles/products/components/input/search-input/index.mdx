検索キーワードを入力させるためのInputの派生コンポーネントです。検索フォームで検索語句を入力するときに使います。`prefix`は検索アイコン「<FaMagnifyingGlassIcon alt="FaMagnifyingGlassIcon 虫眼鏡のアイコン" />」に固定されています。  
[よくあるテーブルのオブジェクトの検索](/products/design-patterns/smarthr-table/#h4-4)などに使用します。

<ComponentStory name="SearchInput" />

## 使用上の注意

### tooltipMessageで入力内容を補足する

`tooltipMessage`を使用して入力内容に対する説明を補足してください。
