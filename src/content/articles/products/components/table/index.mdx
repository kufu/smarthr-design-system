## アクセシビリティ

### 行を選択するチェックボックスやラジオボタンのラベル

`<TdCheckbox>`や`<TdRadioButton>`を用いて`aria-labelledby`で行のオブジェクトを識別するためのテキストを参照してください。

```tsx
<tr key={key}>
  <TdCheckbox aria-labelledby={object.id} />
  <Td>
    <a href={`/hoge/${object.id}`} id={object.id}>{name}</a>
  </Td>
  ...
</tr>
```
