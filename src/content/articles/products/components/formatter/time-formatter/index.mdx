ISO 8601形式の時刻データを任意の形式にフォーマットして表示するコンポーネントです。時刻を[ユーザーの言語や地域設定に適した形式](/products/contents/idiomatic-usage/count/#h4-0)で表示するときに使います。

`useIntl`フックの`formatTime`関数を使用して、時刻をフォーマットできます。

```tsx editable noIframe withStyled
const TimeFormatterExample = () => {
  const { formatTime } = useIntl()
  const date = new Date(2026, 4, 18, 14, 30, 0)
  
  return (
    <time dateTime={date.toISOString()}>
      {formatTime({ date })}
    </time>
  )
}

render(<TimeFormatterExample />)
```

## 使用上の注意

### 表示する時刻のパーツをカスタマイズする

`parts`オプションを使って、表示する時刻のパーツを指定できます。デフォルトは`['hour', 'minute']`（時と分）です。

```tsx editable noIframe withStyled
const TimePartsExample = () => {
  const { formatTime } = useIntl()
  const date = new Date(2026, 4, 18, 14, 30, 45)
  
  return (
    <Stack gap={0.5}>
      <div><strong>デフォルト（時：分）：</strong> 
        <time dateTime={date.toISOString()}>
          {formatTime({ date })}
        </time>
      </div>
      <div><strong>秒を含む（時：分：秒）：</strong> 
        <time dateTime={date.toISOString()}>
          {formatTime({ date, parts: ['hour', 'minute', 'second'] })}
        </time>
      </div>
      <div><strong>時のみ：</strong> 
        <time dateTime={date.toISOString()}>
          {formatTime({ date, parts: ['hour'] })}
        </time>
      </div>
    </Stack>
  )
}

render(<TimePartsExample />)
```

### 12時間形式で表示する

`options`の`hour12`を`true`に設定すると、12時間形式で表示できます（ロケールによって表示形式は異なります）。

```tsx editable noIframe withStyled
const Hour12Example = () => {
  const { formatTime } = useIntl()
  const date = new Date(2026, 4, 18, 14, 30, 0)
  
  return (
    <Stack gap={0.5}>
      <div><strong>24時間形式：</strong> 
        <time dateTime={date.toISOString()}>
          {formatTime({ date })}
        </time>
      </div>
      <div><strong>12時間形式：</strong> 
        <time dateTime={date.toISOString()}>
          {formatTime({ date, options: { hour12: true } })}
        </time>
      </div>
    </Stack>
  )
}

render(<Hour12Example />)
```
