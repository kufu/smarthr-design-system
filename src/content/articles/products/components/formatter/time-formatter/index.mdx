時刻データを任意の形式にフォーマットして表示するコンポーネントです。時刻を[ユーザーの言語や地域設定に適した形式](/products/contents/idiomatic-usage/count/#h4-0)で表示するときに使います。

<ComponentStory name="TimeFormatter" />

## 使用上の注意

### 時刻をフォーマットする機能だけ必要な場合

TimeFormatterは時刻を`time`要素としてレンダリングします。  
フォーマットされた時刻の文字列だけが必要な場合は、`useIntl`の`formatTime`を使用してください。

```tsx editable noIframe withStyled
const FormatTimeExample = () => {
  const { formatTime } = useIntl()
  return (
    <span>現在時刻は{formatTime({ date: new Date(2026, 4, 18, 14, 30, 0) })}です。</span>
  )
}

render(<FormatTimeExample />)
```

## 使用例

### 表示する時刻のパーツをカスタマイズする

`parts`オプションを使って、表示する時刻のパーツを指定できます。デフォルトは`['hour', 'minute']`（時と分）です。

```tsx editable noIframe withStyled
const TimePartsExample = () => {
  const date = new Date(2026, 4, 18, 14, 30, 45)

  return (
    <Stack gap={0.5} as="dl">
      <Cluster gap={1} align="center">
        <Text styleType="blockTitle" as="dt">デフォルト（時：分）</Text>
        <dd>
          <TimeFormatter date={date} />
        </dd>
      </Cluster>
      <Cluster gap={1} align="center">
        <Text styleType="blockTitle" as="dt">秒を含む（時：分：秒）</Text>
        <dd>
          <TimeFormatter date={date} parts={['hour', 'minute', 'second']} />
        </dd>
      </Cluster>
      <Cluster gap={1} align="center">
        <Text styleType="blockTitle" as="dt">時のみ</Text>
        <dd>
          <TimeFormatter date={date} parts={['hour']} />
        </dd>
      </Cluster>
    </Stack>
  )
}

render(<TimePartsExample />)
```

### 12時間形式で表示する

`options`の`hour12`を`true`に設定すると、12時間形式で表示できます（ロケールによって表示形式は異なります）。

```tsx editable noIframe withStyled
const Hour12Example = () => {
  const date = new Date(2026, 4, 18, 14, 30, 0)

  return (
    <Stack gap={0.5} as="dl">
      <Cluster gap={1} align="center">
        <Text styleType="blockTitle" as="dt">24時間形式</Text>
        <dd>
          <TimeFormatter date={date} />
        </dd>
      </Cluster>
      <Cluster gap={1} align="center">
        <Text styleType="blockTitle" as="dt">12時間形式</Text>
        <dd>
          <TimeFormatter date={date} options={{ hour12: true }} />
        </dd>
      </Cluster>
    </Stack>
  )
}

render(<Hour12Example />)
```
