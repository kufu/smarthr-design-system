ISO 8601形式の日付と時刻データを任意の形式にフォーマットして表示するコンポーネントです。日付と時刻を[ユーザーの言語や地域設定に適した形式](/products/contents/idiomatic-usage/count/#h4-0)で表示するときに使います。

`useIntl`フックの`formatTimestamp`関数を使用して、日付と時刻を組み合わせてフォーマットできます。

```tsx editable noIframe withStyled
const TimestampFormatterExample = () => {
  const { formatTimestamp } = useIntl()
  const date = new Date(2026, 4, 18, 14, 30, 0)
  
  return (
    <time dateTime={date.toISOString()}>
      {formatTimestamp({ date })}
    </time>
  )
}

render(<TimestampFormatterExample />)
```

## 使用例

### 表示する時刻のパーツをカスタマイズする

`timeParts`オプションを使って、表示する時刻のパーツを指定できます。デフォルトは`['hour', 'minute']`（時と分）です。

```tsx editable noIframe withStyled
const TimestampPartsExample = () => {
  const { formatTimestamp } = useIntl()
  const date = new Date(2026, 4, 18, 14, 30, 45)
  
  return (
    <Stack gap={0.5} as="dl">
      <Cluster gap={1} align="center">
        <Text styleType="blockTitle" as="dt">デフォルト（時：分）</Text> 
        <dd>
          <time dateTime={date.toISOString()}>
            {formatTimestamp({ date })}
          </time>
        </dd>
      </Cluster>
      <Cluster gap={1} align="center">
        <Text styleType="blockTitle" as="dt">秒を含む（時：分：秒）</Text> 
        <dd>
          <time dateTime={date.toISOString()}>
            {formatTimestamp({ date, timeParts: ['hour', 'minute', 'second'] })}
          </time>
        </dd>
      </Cluster>
      <Cluster gap={1} align="center">
        <Text styleType="blockTitle" as="dt">時のみ</Text> 
        <dd>
          <time dateTime={date.toISOString()}>
            {formatTimestamp({ date, timeParts: ['hour'] })}
          </time>
        </dd>
      </Cluster>
    </Stack>
  )
}

render(<TimestampPartsExample />)
```
