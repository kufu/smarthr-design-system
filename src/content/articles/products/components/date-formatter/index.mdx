## 使用上の注意

### TimeFormatterやTimestampFormatterとの使い分け

日付と時刻を扱う機能は用途に応じて使い分けます。

- **DateFormatter**: 日付のみを表示したい場合
- **[TimeFormatter](/products/components/time-formatter/)**: 時刻のみを表示したい場合
- **[TimestampFormatter](/products/components/timestamp-formatter/)**: 日付と時刻を両方表示したい場合

日付と時刻を分けて表示したい場合は、DateFormatterとTimeFormatterを個別に使用することを検討してください。

### 日付をフォーマットする機能だけ必要な場合

DateFormatterは日付を`time`要素としてレンダリングします。  
フォーマットされた日付の文字列だけが必要な場合は、`useIntl`の`formatDate`を使用してください。

```tsx editable noIframe withStyled
const FormatDateExample = () => {
  const { formatDate } = useIntl()
  return (
    <span>今日は{formatDate({ date: new Date(2025, 7 - 1, 10) })}ではありません。</span>
  )
}

render(<FormatDateExample />)
```
