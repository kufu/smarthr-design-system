[Base](/products/components/base/)や[Dialog](/products/components/dialog/)の内部で視覚的に要素をグルーピングするコンポーネントです。Base内やダイアログコンテンツ内でコンテンツを囲んで「[ブロック](/products/design-patterns/visual-grouping/#h3-6)」領域として示すときに使います。

<ComponentStory name="BaseColumn" />

## デザインパターン

### メッセージを表示するパターン

注意事項や補足事項などを見出しと内容のセットで伝えるパターンです。Baseや各種Dialogの上など、[InformationPanel](/products/components/information-panel/)が配置できない場合に主に使います。

```tsx editable
<Base padding={1.5}>
  <BaseColumn>
    <Stack gap={1}>
      <Heading type="blockTitle" icon={<WarningIcon />}>
        注意事項
      </Heading>
      <p>
        注意事項内容テキスト。注意事項内容テキスト。注意事項内容テキスト。
      </p>
    </Stack>
  </BaseColumn>
</Base>
```
