予期しないエラーが発生した場合に表示するコンポーネントです。

メンテナンスやシステム障害などでデータの閲覧・作成・更新が安全に実行できない状況の場合、予期しないエラーが発生したメッセージを表示します。

予期しないエラーの場合、対応方法やヘルプセンターへのリンクを丁寧に提示することが重要です。アプリケーションのホームへ戻るリンクも表示してください。
