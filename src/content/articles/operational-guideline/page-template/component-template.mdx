
#### プロダクト

```tsx editable withStyled layout=product
const AwesomeComponent:React.FC<{ children: React.ReactNode }> = ({ children }) => {
  return (

    </Wrapper>
  )
}

// withStyled を渡すと styled-components を扱えるようになります。

const Wrapper = styled(Stack)`

  `}
`
```
