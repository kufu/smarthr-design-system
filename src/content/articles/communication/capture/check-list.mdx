画面キャプチャを作成する際の[作成ガイドライン](/communication/capture/creation-guideline/)に基づくチェックリストです。  
チェックする内容によっては例外があることがあります。詳細についてはリンク先も参照してください。



## 内容の調整

### 利用シーンに近い表現
- [ ] [氏名や日付など、実際に重複しない要素は表記を分散させている](/communication/capture/creation-guideline/#h3-3)
- [ ] [特定の性別、年齢層、カテゴリに表記の偏りがない](/communication/capture/creation-guideline/#h3-3)

### サンプル
- サンプルテキスト
    - [ ] [従業員氏名はユーザー社の従業員に掲載されている氏名を利用している](/communication/sample/text/#h3-0)
    - [ ] [メールアドレスは `{名前.苗字}@example.com` の形式になっている](/communication/sample/text/#h3-0)
    - [ ] [従業員番号は `0064` など半端な数字を使用している](/communication/sample/text/#h3-0)
    - [ ] [企業名は `株式会社＊＊＊＊＊` になっている](/communication/sample/text/#h3-1)
    - [ ] [住所、電話番号はデザインシステムで推奨する表記になっている](/communication/sample/text/#h3-2)
- サンプル画像
    - [ ] [プロフィール画像はモデル写真を使用している](/communication/capture/creation-guideline/#h4-1)


## 見た目の調整

### ウィンドウサイズ
#### パソコンの場合
- 横幅
    - [ ] [横幅は1400pxで作成している](/communication/capture/creation-guideline/#h3-5)
- 縦幅
    - [ ] [ページの最上部から最下部まで、全体を収めている](/communication/capture/creation-guideline/#h4-3)
    - [ ] [縦幅は880px以上で作成している](/communication/capture/creation-guideline/#h5-1)
    - [ ] [機能を象徴する重要な要素は、最上部から880px以内に収めている](/communication/capture/creation-guideline/#h5-0)

#### スマートフォンの場合
- 横幅
    - [ ] [横幅は375pxで作成している](/communication/capture/creation-guideline/#h3-5)
- 縦幅
    - [ ] [ページの最上部から最下部まで、全体を収めている](/communication/capture/creation-guideline/#h4-3)
    - [ ] [縦幅は760px以上で作成している](/communication/capture/creation-guideline/#h5-1)
    - [ ] [機能を象徴する重要な要素は、最上部から760px以内に収めている](/communication/capture/creation-guideline/#h5-0)

### フォント
- [ ] [可読性が保たれた読みやすいウェイトになっている](/communication/capture/creation-guideline/#h3-6)

### 色
- [ ] [視認性が保たれた読みやすい色になっている](/communication/capture/creation-guideline/#h3-7)
