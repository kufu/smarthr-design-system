SmartHRサービス全体で利用するアイコンです。


## 利用者・利用範囲

SmartHR Design Systemはサイトにアクセスするすべての人が、公開されているすべてのコンテンツを参照できます。ただし利用に関しては範囲を定めています。  
アイコンの利用者・利用範囲は以下のとおりです。

| 利用者 | 名義 | 利用可否 |
| :-- | :-- | :-- |
| **SmartHR従業員** | **サービスSmartHR・<br>株式会社SmartHR** | **◯ 利用できます** |
| **SmartHR従業員** | **SmartHR Plus β版** | **◯ 利用できます** |
| SmartHR従業員 | SmartHRグループ会社・<br>提供サービス | × 利用できません |
| **SmartHR外部パートナー** | **サービスSmartHR・<br>株式会社SmartHR** | **◯ 利用できます** |
| **SmartHR Plusパートナー** | **SmartHR Plusパートナーが<br>提供するアプリ** | **◯ 利用できます** |
| SmartHRグループ会社従業員 | SmartHRグループ会社・<br>提供サービス | × 利用できません |
| すべてのかた・広報担当者のかた | すべて | × 利用できません |

ほかのコンテンツの利用範囲は[利用者のかたへ](/introduction/user/)を参照してください。


## 利用ガイドライン
- SmartHRに関するシーンでどこでも使用できます。
- solidとoutlineの2種類があります。シーンに合わせて適したほうを選んでください。
![outlineのアイコンとsolidのアイコン](./images/outline-solid.png)
- 基本的に、視認性の高いoutlineを使用してください。
- solidとoutlineの混在は避けてください。
